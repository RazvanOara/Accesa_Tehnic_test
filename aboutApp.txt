        The app is a gamification of swimming. You can accept challenges and complete them,
earning a number of tokens named "Goggles" and earn 25 points each time you complete one. 
      Challenges can be created only by the top10 players sorted by points.In order to create one,
 a player will have to play 10 tokens. Challenges are comlpeted only by trust, however it can be
 done by uploading files to the application, in future updates, such as ".fit". 
        A player can earn Badges, for a specific actions, like: create challenges, add one friend,
 complete challenges.
       The user cand add to friend list other players, however is more like a "folowing" list.
